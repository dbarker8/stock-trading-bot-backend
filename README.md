# stock-trading-bot-backend

This is the Firebase cloud functions backend codebase. For more information see the article at https://medium.com/@deanbarker44/how-i-built-a-profitable-stock-trading-bot-using-javascript-29d9720547db?sk=3bd25e4a12789942655b3ca39761b124

This project uses TypeScript mainly just for async/await syntax, and the Typescript language features are not used.

This is a public version of the repo, and does not include all the necessary credentials and instructions to get up and running. If you are interested in running this project locally, please contact dean@dbrsoftware.com
